# CampusSafeHer



## Problem Statement



Female university students may face safety concerns when traveling alone, returning to hostels late, or requiring immediate assistance during emergencies. Existing communication methods are often slow and unorganized, making it difficult to request help, report incidents, or access reliable safety information.



## Solution Overview



CampusSafeHer is a full-stack web application designed to improve student safety by providing emergency SOS alerts, incident reporting, trusted contact management, an AI-powered safety assistant, and analytics for monitoring reported incidents.
